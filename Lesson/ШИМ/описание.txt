Регистр CCP1CON:
	bit 7-6 Unimplemented: Read as '0'
	-----------------------------------------------------
	bit 5-4 CCPxX:CCPxY:менее значимые биты
	Режим захвата:
	неиспользуемый
	Режим сравнения:
	неиспользуемый
	Режим PWM:
	Эти биты представляют собой два LSbs рабочего цикла PWM. Восемь MSbs находятся в CCPRxL.
----------------------------------------------------------
	bit 3-0 CCPxM3:CCPxM0:Коды выбора режима CCPx
		0000 = Capture / Compare / PWM отключен (сбрасывает модуль CCPx)
0100 = Режим захвата, каждый падающий фронт
0101 = Режим захвата, каждый нарастающий фронт
0110 = Режим захвата, каждый 4-й передний фронт
0111 = Режим захвата, каждый 16-й передний фронт
1000 = Сравнить режим, установить выход на соответствие (бит CCPxIF установлен)
1001 = Режим сравнения, очистить вывод при совпадении (бит CCPxIF установлен)
1010 = Сравнить режим, сгенерировать прерывание программного обеспечения при совпадении (бит CCPxIF установлен, вывод CCPx
не влияет)
1011 = Режим сравнения, специальное событие триггера (бит CCPxIF установлен, контакт CCPx не изменяется); CCP1
сбрасывает TMR1; CCP2 сбрасывает TMR1 и начинает преобразование A / D (если модуль A / D
включен)
11xx = режим PWM
